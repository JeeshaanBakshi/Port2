The Brainy Scale

Built a digital distance measuring device using an Arduino Uno, I2C ultrasonic sensor, and LCD display. It calculates and displays distance to nearby objects in real time.

🛠 Hardware: Arduino Uno, Ultrasonic Sensor (HC-SR04), I2C LCD Display

🌐 Functionality:

  • The ultrasonic sensor emits sound waves and measures the time for the echo to return.
  • Using speed of sound and time-of-flight, the distance is calculated precisely.
  • The result is displayed on the LCD screen for real-time monitoring.
